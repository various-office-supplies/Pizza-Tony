# Pizza Tony GPT

<img width="269" height="59" alt="hey_fellas" src="https://github.com/user-attachments/assets/e8a72925-3727-402b-b99e-91bf01f594ad" />  <br>

Discord app that lets you talk to our favorite goofball... forever.

### Specs:
- Trained for 48 hours using 10k+ discord messages using [GPT-2](https://github.com/openai/gpt-2).
- Original messages retrieved from Discord using [DiscordChatExporter](https://github.com/Tyrrrz/DiscordChatExporter).

### Requirements:
- NodeJS, 16.10.0+
- Eris, 0.17.2+
